# 🐟 AquaCalc — Калькулятор параметров воды

Калькулятор минерализации воды для аквариума (L-сомы и другие виды).

## 🚀 Использование

Просто откройте `index.html` в браузере или посетите сайт: [https://AQUACALC/](https://lamerspy.github.io/Aquacalc/)

## 📋 Функционал

- ✅ Расчёт дозировок солей (Ca, Mg, Na, K)
- ✅ Учёт ионного состава воды
- ✅ Расчёт TDS и жёсткости в CaCO₃
- ✅ Сохранение настроек в браузере
- ✅ Экспорт отчёта в HTML
